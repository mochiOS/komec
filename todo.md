## 基礎構文のCodegen拡張

1行の変数宣言だけでなく、基本的な計算や関数呼び出しを LLVM IR に変換できるようにする。

* [ ] **演算子 (`Expr::BinaryOp`) の実装**
* [x] `+`, `-`, `*`, `/` の数値計算（LLVM の `build_int_add` など）への対応
* [ ] ブール演算（`&&`, `||`）の実装
* [ ] 比較演算（`==`, `!=`, `<`, `>`, `<=`, `>=`）の実装
* [ ] ビット演算（`&`, `|`, `^`）の実装
* [ ] 単項演算（`-` や `!`）の実装
* [ ] `in`, `??` などの特殊な演算子の実装
* [ ] 演算子の優先順位と結合規則の正しい処理
* [ ] 演算子オーバーロードのサポート（例: `+` を文字列結合にも使えるようにする）

* [ ] **関数・メソッド呼び出し (`Expr::CallChain`) の実装**

* [ ] `if`文とか`match`とか`for`とかの制御構文を実装する
* [ ] `fn`で関数を定義して呼び出せるようにする

## `state`と`within`の実装

リアクティブな範囲制限変数を実装する。

* [ ] **`range_limit` (`within ~ cycle/clamp`) の Codegen 設計**
* [ ] 変数への代入（`+= 1` や `-= 1`）をパースできるようにグラマーと AST を拡張
* [ ] 代入時に LLVM の `br`（条件分岐命令）と Basic Block を使って境界チェックを実装
* [ ] `cycle`: 上限を超えたら `0` に戻すロジック
* [ ] `clamp`: 上限で値を止めるロジック（`min`/`max` 処理）
